# Sudoku Solver

## Problem

Given a partially filled 9×9 Sudoku grid, find a valid solution.

## What to do

- Implement a backtracking algorithm to solve Sudoku puzzles.
- Print the completed grid.
- Test on easy, medium, and hard puzzles.

## Analysis Focus

How does backtracking work? What is the worst-case time complexity, and why is it usually much faster in practice?

## Deliverables

- GitHub Repository
- Source Code
    - Clean, commented code
- README.md
    - Name, email, and project title
    - Short description of the problem and your approach
    - Instructions on how to run your program
    - A link to your pre-recorded video presentation (see below)
    - AI Usage Declaration: Briefly describe how you used AI tools
    - A short written analysis
        - What problem are you solving?
        - How does your algorithm work?
        - What is the time complexity and why?
- Test Cases
    - At least 3 test inputs with expected outputs
    - Include at least edge case
